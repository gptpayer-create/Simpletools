Frequently Asked Questions

What is the difference between grayscale and black-and-white?
Grayscale means each pixel has a single luminance value ranging from black to white, with many shades of gray in between. True "black and white" is binary — only fully black or fully white. This tool creates grayscale images, which preserve the full tonal range of your photo.
Which channel mix mode should I choose?
Luminance (Perceptual) is recommended for most photos — it weights the red, green and blue channels (0.299R + 0.587G + 0.114B) to match how the human eye perceives brightness. Average is simpler but can look flat. Red or Green channels can give interesting artistic results for portraits and landscapes respectively.
Does this reduce file size?
Saving as PNG grayscale does not significantly reduce size since PNGs store full color info. For smaller files, choose JPG or WEBP output format. JPG at default quality will be considerably smaller than the original PNG.
Is my image uploaded to a server?
No. This tool is entirely client-side. Your image is processed in your browser using the HTML5 Canvas API and never sent to any server. Your photos remain completely private.

Image Grayscale Tool — Complete Guide

Converting images to grayscale removes all color information, rendering the photo in shades of black and white. It's used in photography, document scanning, printing, and creating artistic black-and-white effects.

🎞️ Artistic photography

Black-and-white photos have a timeless, classic aesthetic. Convert color photos to grayscale to create dramatic portraits and documentary-style images.

🖨️ Reduce ink usage

Printing in grayscale uses only black ink, significantly reducing printing costs and ink consumption for documents and internal reports.

📄 Document scanning

Scanned documents with colored backgrounds or highlights can be converted to grayscale for cleaner, more readable black-and-white output.

💻 Technical applications

Machine learning and computer vision algorithms often work with grayscale images as input, requiring conversion before model processing.

Grayscale conversion calculates a weighted average of the red, green, and blue channels of each pixel — typically using the luminance formula (0.299R + 0.587G + 0.114B) to account for human eye sensitivity to different colors. This results in a perceptually accurate brightness representation rather than a simple average.
