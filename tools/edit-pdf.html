Frequently Asked Questions

Can I edit existing text in the PDF?
This tool adds annotations on top of the existing PDF content. To replace existing text, use the redact tool to black it out, then add new text on top.
Is my document kept private?
Yes — all editing happens 100% in your browser. Your PDF is never uploaded to any server.
Does it support multi-page PDFs?
Yes! All pages appear in the sidebar. You can navigate between pages and annotate each one independently.
Are my annotations saved when I download?
Yes. All annotations (text, drawings, highlights, shapes, redactions) are embedded into the downloaded PDF using pdf-lib.
