Frequently Asked Questions

Does dark mode change the actual PDF?
No. The dark mode is applied visually using canvas filters in your browser. The original PDF file is never modified. Exported pages are dark-mode PNG screenshots, not a new PDF.
Can I export a dark-mode version as a new PDF?
You can export individual pages as dark PNG images. For a full dark-mode PDF, use our PDF to B&W tool with the Inverted mode and download the result.
What's the difference between Night Red and Night Blue?
Night Red overlays a warm red-amber tint which is less disruptive to your night vision — great in pitch-dark rooms. Night Blue applies a cooler blue-grey tone, similar to many OS dark modes.
Are my files uploaded anywhere?
Never. All PDF rendering is done locally in your browser using PDF.js. Your file never leaves your device.
