Frequently Asked Questions

Does this support all SQL dialects?
The formatter handles standard ANSI SQL and has dialect modes for MySQL, PostgreSQL, SQLite, and SQL Server. Dialect-specific functions and keywords are highlighted appropriately.
Can I format multiple SQL statements at once?
Yes. Paste multiple statements separated by semicolons and each will be formatted independently with proper line breaks between them.
Is my SQL sent to any server?
No. All formatting happens client-side in your browser using JavaScript. Your queries never leave your device — safe for sensitive database schemas.
