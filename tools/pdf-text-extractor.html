Frequently Asked Questions

Does this work on scanned PDFs?
Only text-based PDFs are supported. Scanned documents are images and require OCR software. If you get no text, your PDF is likely a scanned image.
Will the formatting be preserved?
Basic paragraph structure is preserved. Complex layouts like tables and columns may not extract perfectly, as PDFs store text in reading-order streams rather than visual position.
What's the Markdown download for?
The MD download adds page headers (## Page 1, ## Page 2, etc.) making it easy to navigate the content in any Markdown editor or import into tools like Notion or Obsidian.
