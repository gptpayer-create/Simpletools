Frequently Asked Questions

How many files can I batch convert at once?
There is no hard limit on the number of files. You can add 50, 100, or more files to the queue. Performance depends on your device's available memory and browser. For very large batches (500+ files), we recommend processing in groups of 50–100.
What image formats are supported?
The tool supports JPG, JPEG, PNG, WEBP, GIF, and BMP image formats. It also accepts PDF files for the Merge PDFs and Compress PDFs modes. HEIC/HEIF files from iPhones are not yet supported — convert them to JPG first.
Does "Compress PDF" actually reduce file size?
Yes. The compress mode re-renders each page of the PDF as an image at a lower quality setting and re-embeds it, which significantly reduces file size for image-heavy PDFs. Text-only PDFs may see less reduction since the text is vector-based and already compact.
Can I reorder the files before converting?
Yes. The file queue shows your files in the order they will be processed. Use the Sort by Name or Reverse Order buttons to reorder the queue. The final PDF will respect this order, which is especially important when merging multiple images into a single document.
Are my files uploaded to a server?
Absolutely not. All conversion happens locally in your browser using pdf-lib and the HTML5 Canvas API. Your images and documents never leave your device. This makes the tool completely safe for confidential or sensitive files.
