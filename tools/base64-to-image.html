Frequently Asked Questions

What formats can I decode?
You can decode any image type encoded as Base64, including PNG, JPEG, GIF, WEBP, SVG, and BMP. Simply paste the Data URI (with prefix) or plain Base64 string.
My Base64 string doesn't start with "data:" — will it work?
Yes! If you paste plain Base64 without a Data URI prefix, the tool will attempt to auto-detect the image format. For best results, include the full Data URI prefix like data:image/png;base64,.
Why is there a checkerboard background in the preview?
The checkerboard pattern indicates transparent areas in PNG images. This is standard practice to visualize transparency, similar to how Photoshop displays it.
