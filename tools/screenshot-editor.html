Frequently Asked Questions

Can I paste a screenshot directly from clipboard?
Yes! Press Ctrl+V (or Cmd+V on Mac) anywhere on the page and your clipboard screenshot will load directly into the editor without needing to save it as a file first.
How many steps of undo does it support?
The editor keeps a full history of every drawing action. You can undo all the way back to the original image using Ctrl+Z repeatedly, or the ↩ button in the toolbar.
Can I change the color and thickness of annotations?
Yes. Use the color pickers in the toolbar to set stroke and fill colors. The size dropdown controls line width. Font size can be set for the text tool separately.
Is the output lossless?
Output is saved as PNG which is a lossless format. All annotations are composited at full source resolution before download, so quality is preserved exactly.
Does it work on mobile devices?
Yes, all drawing tools support touch input. Tap and drag to draw on mobile. The toolbar scrolls horizontally on small screens.
